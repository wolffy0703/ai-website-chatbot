# AI Website Chatbot

A simple website chatbot with a Python backend that answers FAQs or context-based questions.

## Features
- Minimal chat UI (HTML/CSS/JS)
- Python backend (Flask or FastAPI)
- FAQ-driven responses or AI integration

## Tech Stack
- Python (Flask / FastAPI)
- HTML / CSS / JavaScript

## Quick Start
1. `pip install -r requirements.txt`
2. `python src/app.py`
3. Open `http://localhost:5000` in your browser
